# Additional Icons

This directory contains additional open-source SVG icons for use in Tool3.

## Heroicons

**Source**: [Heroicons by Tailwind Labs](https://github.com/tailwindlabs/heroicons)  
**License**: MIT License  
**Icons**: ~300+ outline icons

Heroicons are a set of free MIT-licensed SVG icons optimized for 24x24px. They're perfect for UI elements and educational visuals.

### Usage

Icons are automatically loaded by the backend and merged with the math2visual dataset icons. They appear in the Search Icons panel with the prefix `heroicon-` (e.g., `heroicon-academic-cap`, `heroicon-book`, etc.).

### Adding More Icons

To add more icon sets:

1. Download SVG icons to this directory
2. Ensure they're named descriptively (e.g., `book.svg`, `calculator.svg`)
3. Restart the backend server
4. Icons will automatically appear in the Search Icons panel

### Popular Open-Source Icon Sets

- **Heroicons**: Already included (MIT License)
- **Lucide Icons**: https://lucide.dev/icons/ (ISC License)
- **Material Design Icons**: https://materialdesignicons.com/ (Apache 2.0)
- **Feather Icons**: https://feathericons.com/ (MIT License)

All icons should be in SVG format for best compatibility.
